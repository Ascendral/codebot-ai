Self-Review Gate

Agent reviews its own changes with an independent prompt. Three verdicts: APPROVE, REVISE, or REJECT. No unreviewed code ships.

Audit Trail

Every tool invocation, file edit, test result, and LLM decision logged as structured JSON. Full traceability from issue to merged PR.
