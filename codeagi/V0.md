# V0.md

## Purpose

V0 is not AGI.

V0 is the first honest milestone for CodeAGI:
a persistent autonomous agent runtime with explicit internal state, task continuity, memory, planning, execution, and reflection.

## V0 Goal

Build a system that can:
- accept a mission
- break it into tasks
- maintain working state
- persist across sessions
- execute actions
- reflect on outcomes
- update memory
- continue progress over time

## V0 Must Include

1. Mission intake
2. Task graph
3. Working memory
4. Long-term memory
5. World state
6. Planner
7. Action execution
8. Reflection
9. Persistence
10. Evaluation harness

## V0 Success Criteria

V0 is successful if it can reliably:
1. resume an interrupted mission
2. complete multi-step work with limited intervention
3. store prior experience and use it later
4. detect failed steps and replan
5. produce structured records of what it did and why
6. improve repeated-task performance when prior memory is available

## Honest Claim After V0

If V0 works, the honest claim is:

CodeAGI is a persistent autonomous agent runtime with explicit memory, planning, state continuity, and structured reflection.
